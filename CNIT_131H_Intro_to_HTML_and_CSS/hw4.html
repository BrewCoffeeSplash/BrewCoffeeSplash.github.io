Lists Usage
Unordered List An unordered list is used for a list of items that don't have to be in any particular order. To start this list use the tag <ul>.
Ordered List An ordered list is used for a list of items that have to be in a specific order. To start this list use the tag <ol>.
Definition List An definition list is used for pairs of items that are the term defined and it's definition. To start this list use the tag <dl>.

FTP
File Transfer Protocol. Network protocol for transferring files from one computer to another on a TCP/IP network.
SSH
Secure Shell. Cryptographic network protocol used to provide a secure channel over an unsecured network. A user can access a computer securely over unsecured networks like publicly free wifi to their home computer/server.
