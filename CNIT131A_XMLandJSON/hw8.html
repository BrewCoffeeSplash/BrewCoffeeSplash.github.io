XQuery Report by William Sawyer

  • What is the main characteristic of XQuery?
    • It's similar to SQL or MongoDB in querying data but can also query data from multiple data sources. It's a expression-oriented programing language with a simple type system.(1)
  • Look at the XML in hw8.xml file (you will find this file in the CNIT131AFiles.zip folder you downloaded for this course). Look at the code in hw8.xq and based on your research, what is this code doing if applied on the XML of hw8.xml? Explain this example within your report
    • It looks the the XML file hw8.xml and goes to the book element of the root element bookstore. Under the price element of the book element, it then looks for any that have the price element that is less than 30. It returns the resulting title element for each book.
  • What exactly is XQIB? How it can be used with XQuery?
    • XQIB allows you to execute Xquery code like you would use Javascript. You can change the DOM, add event listeners, and make REST calls. (2,3)
